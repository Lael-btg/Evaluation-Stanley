Pourquoi HTML

Le HTML (HyperText Markup Language) est le langage de balisage utilisé pour créer la structure de base d'une page web. Il définit les éléments tels que les titres, paragraphes, liens et images. Exemple de code HTML : 

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
                <p>This is a simple HTML page.</p>
            </body>
            </html>
Ceci est un exemple de code HTML qui crée une page web simple avec un titre et un paragraphe.

Balises HTML

Les balises HTML sont utilisées pour structurer le contenu de la page, tandis que les éléments tels que les titres et les paragraphes sont définis à l'aide de balises spécifiques. "H1" est utilisé pour les titres principaux, tandis que "p" est utilisé pour les paragraphes. la balise <ul></ul> est utilisée pour créer une liste non ordonnée, tandis que la balise <li> est utilisée pour définir chaque élément de la liste. les balises <a> sont utilisé pour créer des liens hypertextes, permettant aux utilisateurs de naviguer vers d'autres pages ou ressources en ligne. les balises <img> sont utilisées pour insérer des images dans une page web, en spécifiant l'attribut "src" pour indiquer la source de l'image et "alt" pour fournir une description alternative. Il faut voir le HTML comme le squelette du corps humain, fournissant la structure de base.

qu'est ce que le CSS

Le CSS (Cascading Style Sheets) est le langage de style utilisé pour définir l'apparence et la mise en forme d'une page web. Il permet de contrôler la disposition, les couleurs, les polices et autres aspects visuels. Si le HTML est le squelette d'une page web, le CSS en est la peau donnant vie à la structure de base. le CSS ne peut pas s'utiliser seul, il doit être associé au HTML pour donner vie à la structure de base d'une page web. En combinant le HTML pour la structure et le CSS pour la présentation, vous pouvez créer des sites web attrayants et fonctionnels.

Comment utiliser le CSS

Il existe plusieurs façons d'utiliser le CSS pour styliser une page web. Vous pouvez inclure des styles CSS directement dans les éléments HTML à l'aide de l'attribut "style", ou vous pouvez créer un fichier CSS externe et le lier à votre page HTML à l'aide de la balise <link> dans la section <head> de votre document. Exemple d'utilisation du CSS en ligne : 

            <h1 style="color: blue; font-size: 24px;">Hello, World!</h1>
Exemple d'utilisation du CSS externe : 
            <head>
                <link rel="stylesheet" href="styles.css">
            </head>
Dans cet exemple, nous avons créé un fichier CSS externe nommé "styles.css" et nous l'avons lié à notre page HTML. Ensuite, nous pouvons définir nos règles de style dans ce fichier CSS pour personnaliser les éléments de notre page web.

tips et bonne pratique

Pour une meilleur expérience utilisateur, il est important de suivre certaines bonnes pratiques lors de la création de pages web avec HTML et CSS. Voici quelques conseils utiles :

Utilisez des balises sémantiques pour structurer votre contenu de manière logique et compréhensible.

Organisez votre code CSS de manière claire et maintenable en utilisant des classes et des ID pour cibler les éléments spécifiques.

Evitez au maximum les <div> sans sens sémantique.
